# Data Structures & Algorithms

Personal DSA practice repo in **C++** — following [Striver's A2Z DSA Sheet](https://takeuforward.org/strivers-a2z-dsa-course/strivers-a2z-dsa-course-sheet-2/) (474 problems) alongside the 100xSchool Bootcamp.

See [ROADMAP.md](./ROADMAP.md) for the full phase-wise plan.

***

## Structure

```
.
├── 01-foundations/          # C++ STL, syntax drills, language warmups
├── 02-data-structures/      # DS built from scratch (arrays, linked list, trees...)
├── 03-algorithms/           # Algorithm concepts and implementations
├── 04-problems/             # Freeform practice problems (LeetCode, GFG, CF...)
└── 05-bootcamp/             # 100xSchool Bootcamp 1.0 — weekly lectures + contests
```

***

## 03-algorithms

| Topic        | Files                                                                           |
| ------------ | ------------------------------------------------------------------------------- |
| `hashing/`   | character hashing, number hashing, optimized hashing, highest occurring element |
| `recursion/` | fibonacci, factorial, palindrome check, reverse array, sum of N numbers         |
| `sorting/`   | bubble sort                                                                     |
| `kadane/`    | kadane's algorithm (max subarray sum)                                           |

## Conventions

* All files: `kebab-case.cc`
* Bootcamp assignment files keep their letter prefix (`a-`, `b-`, `c-`...) for ordering
* No compiled artifacts committed (`*.exe`, `*.o`, `*.out` are gitignored)
